How to run Agrico
1. Download and Unzip file on your local system copy Agrico.
2. Put that downloaded project inside XAMPP/htdoc folder which is present in C-Drive.
3. Downlaod Xammp Server
4. Database Configuration
   Open PHPMyAdmin
   Create Database agrico
   Import database agrico.sql (available inside zip package)
5. For Admin Panel Open Your browser put inside browser “http://localhost/agrico/agrico/admin”
